│   │   ┌─────────────────────────────────────────────────────────────────────────────────────────────┐   │   │
│   │   │  5. 边界框细化 (Bounding Box Refinement)                                                    │   │   │
│   │   │     delta_bbox = MLP(queries'''')  → [2, 900, 4]                                            │   │   │
│   │   │     new_ref_points = sigmoid(logit(ref_points) + delta_bbox)                                │   │   │
│   │   │                                                                                               │   │   │
│   │   │     输出: refined_reference_points: [2, 900, 4]                                             │   │   │
│   │   └─────────────────────────────────────────────────────────────────────────────────────────────┘   │   │
│   │                                                                                                      │   │
│   │  重复6层 (decoder_layers = 6)                                                                        │   │
│   │  每层输出: queries [2, 900, 256] + refined_ref_points [2, 900, 4]                                   │   │
│   └─────────────────────────────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                                               │
│   ┌─────────────────────────────────────────────────────────────────────────────────────────────────────┐   │
│   │                          解码器输出 (Decoder Outputs)                                               │   │
│   │                                                                                                      │   │
│   │  last_hidden_state: [2, 900, 256]                 (最后一层输出)                                     │   │
│   │  intermediate_hidden_states: [2, 6, 900, 256]    (所有层输出)                                       │   │
│   │  intermediate_reference_points: [2, 6, 900, 4]   (所有层参考点)                                     │   │
│   └─────────────────────────────────────────────────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────────────────────────────────────────────────┘

                                                    ▼

┌─────────────────────────────────────────────────────────────────────────────────────────────────────────────┐
│                                    预测头 (Prediction Heads)                                                 │
├─────────────────────────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                                               │
│   ┌─────────────────────────────────────────────────────────────────────────────────────────────────────┐   │
│   │  分类头 (Classification Head)                                                                       │   │
│   │                                                                                                      │   │
│   │  for each decoder layer i:                                                                          │   │
│   │    logits_i = ContrastiveEmbedding(                                                                │   │
│   │        vision_hidden=intermediate_hidden_states[:, i],  # [2, 900, 256]                             │   │
│   │        text_hidden=encoder_text_features,                 # [2, 10, 256]                            │   │
│   │        text_mask=attention_mask                          # [2, 10]                                  │   │
│   │    ) → [2, 900, num_classes]                                                                        │   │
│   │                                                                                                      │   │
│   │  矩阵运算:                                                                                           │   │
│   │    logits = V @ T^T  (where V: [2, 900, 256], T: [2, 10, 256])                                     │   │
│   │    → [2, 900, 10] → masked → padded to [2, 900, max_text_len]  最像什么词                                    │   │
│   │                                                                                                      │   │
│   │  最终: logits_all = [6, 2, 900, num_classes]  (6层, 2 batch, 900 queries)  每一层的都是不一样的                        │   │
│   └─────────────────────────────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                                               │
│   ┌─────────────────────────────────────────────────────────────────────────────────────────────────────┐   │
│   │  边界框回归头 (Bounding Box Regression Head)                                                        │   │
│   │                                                                                                      │   │
│   │  for each decoder layer i:                                                                          │   │
│   │    delta_bbox = MLP(intermediate_hidden_states[:, i])  # 3层MLP                                    │   │
│   │    → [2, 900, 4]                                                                                    │   │
│   │                                                                                                      │   │
│   │    if i == 0:                                                                                       │   │
│   │        ref = init_reference_points  # [2, 900, 4]                                                   │   │
│   │    else:                                                                                            │   │
│   │        ref = intermediate_reference_points[:, i-1]  # 上一层的参考点                                │   │
│   │                                                                                                      │   │
│   │    pred_boxes_i = sigmoid(logit(ref) + delta_bbox)  # [2, 900, 4]                                  │   │
│   │                                                                                                      │   │
│   │  最终: pred_boxes_all = [6, 2, 900, 4]                                                              │   │
│   └─────────────────────────────────────────────────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────────────────────────────────────────────────┘

                                                    ▼

│                                                                                                               │
│  最终预测:                                                                                                    │
│  ┌─────────────────────────────────────────────────────────────────────────────────────────────────────┐   │
│  │  logits (最后一层): [2, 900, num_classes]                                                            │   │
│  │  pred_boxes (最后一层): [2, 900, 4]                                                                  │   │
│  │                                                                                                      │   │
│  │  示例:                                                                                                │   │
│  │    logits[0, 0, :] = [0.1, 0.8, 0.05, ...]  # 第1张图第1个查询的类别分数                          │   │
│  │    pred_boxes[0, 0, :] = [0.45, 0.55, 0.3, 0.25]  # cx, cy, w, h (归一化)                         │   │
│  └─────────────────────────────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                                               │
│  后处理 (Post-Processing):                                                                                   │
│  ┌─────────────────────────────────────────────────────────────────────────────────────────────────────┐   │

│  │                                                                                                      │   │
│  │  最终结果:                                                                                            │   │
│  │    boxes: [[x1, y1, x2, y2], ...]  # 绝对坐标                                                       │   │
│  │    scores: [0.95, 0.87, ...]      # 置信度分数                                                       │   │
